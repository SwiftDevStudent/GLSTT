GLSTT App Icon Sources

These SVGs are authored for Icon Composer on a 1024x1024 canvas with no background.

Import order, back to front:
- `01_SpeechBubble.svg`
- `02_STTWave.svg`
- `03_STTText.svg`
- `04_CursorAccent.svg`

Notes:
- Keep these as four Icon Composer groups/layers max.
- Let Icon Composer provide the background, material, shadows, and translucency.
- Recommended background:
  - Solid `#F97316`
  - Or a subtle gradient from `#FF9A3D` to `#F25C05`

Concept:
- Speech bubble: speech input
- Waveform: spoken audio
- Text lines + caret: transcript output
